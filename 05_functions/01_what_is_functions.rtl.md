# جزوه آموزشی آشنایی با توابع در PHP

## مقدمه

توابع (Functions) یکی از مهم‌ترین مفاهیم در برنامه‌نویسی هستند که باعث ساده‌تر شدن کدنویسی، قابل‌فهم‌تر شدن پروژه‌ها و افزایش سرعت توسعه می‌شوند. در PHP نیز توابع نقش کلیدی دارند و برنامه‌نویسان در پروژه‌های واقعی به شکل گسترده از آن‌ها استفاده می‌کنند.

در این جزوه ابتدا با مفهوم توابع و مزایای استفاده از آن‌ها آشنا می‌شوید و سپس 20 تابع پرکاربرد PHP را همراه با مثال یاد می‌گیرید.

---

## فواید و کاربرد توابع در PHP

### 1. جلوگیری از تکرار کد (DRY)

توابع کمک می‌کنند بخش‌های تکراری کد را تنها یک بار بنویسیم و هر زمان لازم شد آن را فراخوانی کنیم.

### 2. قابل خواندن شدن و نظم بیشتر

کدی که به بخش‌های کوچک (توابع) تقسیم شده باشد قابل فهم‌تر است.

### 3. دیباگ راحت‌تر

وقتی هر بخش از برنامه در یک تابع باشد، پیدا کردن خطا و اصلاح آن بسیار ساده‌تر می‌شود.

### 4. قابلیت استفاده مجدد (Reusable)

توابع را می‌توان در بخش‌های مختلف پروژه یا حتی پروژه‌های دیگر نیز به‌کار برد.

### 5. تقسیم وظایف

هر تابع یک کار مشخص انجام می‌دهد و برنامه منظم‌تر می‌شود.

---

## ساختار تعریف یک تابع در PHP

```php
function functionName($parameter1, $parameter2) {
    // دستورات تابع
    return $result; // اختیاری
}
```

فراخوانی تابع:

```php
functionName($value1, $value2);
```

---

## انواع توابع در PHP

در PHP دو نوع تابع وجود دارد که هر کدام کاربرد خاص خود را دارند.

### 1. توابع پیش‌ساخته (Built‑in Functions)

توابعی هستند که خود زبان PHP آن‌ها را ساخته و آماده استفاده قرار داده است. این توابع بسیار سریع، بهینه و قابل اعتماد هستند.

#### ویژگی‌ها

* از قبل نوشته شده‌اند
* در تمام پروژه‌ها در دسترس هستند
* دارای دسته‌بندی‌های مختلف‌اند
* معمولاً عملکردی بسیار سریع‌تر از کد نویسی دستی دارند

#### تعداد توابع پیش‌ساخته

- **بیش از 1000 تابع پیش‌فرض** وجود دارد که در حوزه‌های مختلف طبقه‌بندی شده‌اند، مانند:

* توابع رشته‌ای (String Functions)
* توابع آرایه‌ای (Array Functions)
* توابع تاریخ و زمان (Date/Time)
* توابع فایل و دایرکتوری (File System)
* توابع ریاضی (Math Functions)
* توابع مرتبط با پایگاه داده (MySQLi, PDO)
* توابع JSON
* توابع مربوط به سشن‌ها (Session Functions)
* توابع اعتبارسنجی و بررسی متغیرها (Variable Handling)

### مثال توابع پیش‌ساخته

```php
strlen("Hello");
array_push($arr, "item");
json_encode($data);
```

---

### 2. توابع تعریف‌شده توسط کاربر (User‑Defined Functions)

توابعی هستند که برنامه‌نویس خودش آن‌ها را تعریف می‌کند تا کد تمیزتر و قابل‌استفاده‌ی مجدد شود.

#### ویژگی‌ها

* توسط برنامه‌نویس ساخته می‌شوند
* برای سازماندهی کد بسیار مهم‌اند
* می‌توانند ورودی و خروجی داشته باشند

### مثال

```php
function sum($a, $b) {
    return $a + $b;
}

echo sum(5, 7);
```

# 20 تابع پیش ساخته پرکاربرد PHP

در ادامه 20 تابع بسیار مهم و پر استفاده در برنامه‌نویسی واقعی معرفی می‌شود.

---

## 1. isset()

برای بررسی موجود بودن یک متغیر و اینکه مقدار null نباشد.

```php
if (isset($name)) {
    echo "مقدار تنظیم شده است";
}
```

## 2. empty()

برای بررسی خالی بودن متغیر.

```php
if (empty($text)) {
    echo "ورودی خالی است";
}
```

## 3. print_r()

برای نمایش آرایه یا آبجکت.

```php
print_r($array);
```

## 4. var_dump()

نمایش جزئیات کامل نوع و مقدار متغیر.

```php
var_dump($data);
```

## 5. trim()

حذف فاصله‌ها از ابتدا و انتهای رشته.

```php
$name = trim($name);
```

## 6. strlen()

محاسبه طول رشته.

```php
echo strlen("Hello");
```

## 7. str_replace()

جایگزینی رشته.

```php
echo str_replace("world", "PHP", "Hello world");
```

## 8. strtolower() و strtoupper()

تبدیل رشته به کوچک یا بزرگ.

```php
echo strtolower("HELLO");
```

## 9. explode()

تبدیل رشته به آرایه.

```php
$parts = explode(",", "a,b,c");
```

## 10. implode()

تبدیل آرایه به رشته.

```php
echo implode("-", [1,2,3]);
```

## 11. array_push()

افزودن مقدار به انتهای آرایه.

```php
array_push($list, "new item");
```

## 12. in_array()

بررسی حضور مقدار در آرایه.

```php
if (in_array("apple", $fruits)) {
    echo "وجود دارد";
}
```

## 13. count()

تعداد عناصر آرایه.

```php
echo count($students);
```

## 14. json_encode() و json_decode()

تبدیل آرایه به JSON و برعکس.

```php
$json = json_encode($data);
$array = json_decode($json, true);
```

## 15. date()

دریافت تاریخ و زمان.

```php
echo date("Y-m-d H:i:s");
```

## 16. time()

دریافت timestamp.

```php
echo time();
```

## 17. header()

برای تغییر هدر HTTP (مثل ریدایرکت).

```php
header("Location: home.php");
```

## 18. include و require

برای وارد کردن فایل در PHP.

```php
include "header.php";
require "config.php";
```

## 19. isset() در فرم‌ها

کاربرد عملی در پروژه واقعی.

```php
if (isset($_POST['submit'])) {
    echo "فرم ارسال شده";
}
```

## 20. die() / exit()

برای توقف اجرای برنامه.

```php
die("متوقف شد");
```

## جمع‌بندی

توابع یکی از پایه‌های مهم برنامه‌نویسی PHP هستند. با استفاده از توابع:

* کدهایتان منظم‌تر می‌شوند
* خوانایی برنامه بسیار بیشتر می‌شود
* پروژه‌ها سریع‌تر توسعه پیدا می‌کنند
* خطایابی ساده‌تر انجام می‌شود

در این جزوه 20 تابع پرکاربرد معرفی شد که در پروژه‌های واقعی PHP بیشترین استفاده را دارند.
